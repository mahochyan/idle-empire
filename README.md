# 放置帝国 (Idle Empire)

文字挂机 RTS 游戏 — 资源生产 × 军备养成 × 回合制自动战斗

## 当前开发状态

**Phase 1 已完成**

| 模块 | 状态 |
|------|------|
| 资源系统 | 木材/石料/食物，建筑自动产出 |
| 建筑系统 | 伐木场/采石场/农田/营帐，建造+升级计时 |
| 兵种系统 | 步兵/弓兵/骑兵/枪兵/法师 共 5 兵种 |
| 克制环 | 步→枪→骑→弓→步，优势 1.3×/劣势 0.7× |
| 法师体系 | 独立互易伤 1.3×，玻璃大炮定位 |
| 种族被动 | 人族格挡/精灵暴击/兽人吸血/亡灵破甲 |
| 训练系统 | 消耗资源训练士兵，加入军营池 |
| 阵型系统 | 前/中/后排，兵团槽位，人数=HP |
| 编队弹窗 | 选兵种+定人数，长按加减/MAX一键填满 |
| 战斗系统 | 回合制自动演算，文字逐行播报 |
| 战斗动画 | CSS shake/hit/dead 动画 |
| 速度控制 | 1x/2x/4x 三档变速 |
| 敌人系统 | 5 普通敌人 + 1 Boss，JSON 配置，依次解锁 |
| 战后处理 | 存活单位重建阵容 + 军营自动补员 |
| UI 布局 | 手机框体 + 底部 5 标签页导航 |
| 存档 | localStorage 存取 |

## 技术栈

- 原生 HTML/CSS/JS（ES6），单文件 `index.html`
- JSON 配置化：`CFG` 对象包含所有兵种/建筑/资源/敌人数据
- 1 秒 tick 驱动资源产出 + 建筑计时
- 战斗独立状态机 `B`，不影响主状态 `S`

## 核心架构

| 变量 | 用途 |
|------|------|
| `S` | 主状态对象（资源、建筑等级、军营、编队） |
| `B` | 战斗状态机（独立，不影响主状态） |
| `CFG` | 所有配置数据 |
| `tickTimer` | 1 秒定时器 |

## 伤害公式

```
伤害 = 攻击 × 克制倍率 × 法师易伤 × 随机浮动(0.9~1.1) − 防御减免
```

- 克制倍率：优势 1.3 / 普通 1.0 / 劣势 0.7
- 弓兵对非骑兵目标基础 20% miss，对骑兵 50% miss
- 法师易伤（双向）：1.3

放置帝国 — 菜单目录
====================

顶部资源栏
  ├─ 木材 (wood)
  ├─ 石料 (stone)
  ├─ 食物 (food)
  ├─ 科技点 (tech)
  └─ 人口 (pop)

底部导航
  ├─ 主页 (home)
  │   ├─ 城镇信息卡片（等级 / 人口 / 仓库 / Boss 进度 / 升级按钮）
  │   ├─ 人口分配卡片（木材/石料/食物 ± 调节 + 采集速率 + Buff%）
  │   ├─ 城镇巡防地图（建筑 / 工人动画 / 驻军守卫 / 入侵 VFX）
  │   ├─ 最近事件卡片（固定行高，日志倒序 6 条）
  │   ├─ 设置按钮（居中，点击打开设置弹窗）
  │   └─ 测试工具卡片（激活码解锁后显示，一键添加资源）
  │
  ├─ 建筑 (build)
  │   ├─ 基础建筑 (basic)
  │   │   ├─ 营帐 (barracks)
  │   │   ├─ 仓库 (warehouse)
  │   │   ├─ 伐木场 (lumber_mill)
  │   │   ├─ 采石场 (quarry)
  │   │   └─ 农田 (farm)
  │   ├─ 兵营建筑 (barracks)
  │   │   ├─ 步兵营地 (infantry_camp)
  │   │   ├─ 射手靶场 (archer_range)
  │   │   ├─ 骑兵训练场 (stable)
  │   │   ├─ 长矛兵营地 (spear_crypt)
  │   │   └─ 法师塔 (mage_tower)
  │   ├─ 特殊建筑 (special)
  │   │   └─ 箭塔 (arrow_tower)
  │   └─ 英雄祭坛 (altar) · 暂无建筑
  │
  ├─ 军营 (barracks)
  │   └─ 5 兵种卡片 × 每兵种
  │       ├─ 兵种名（可点击 → 单位详情弹窗）
  │       │   └─ 属性面板：攻击力 / 防御力 / 速度 / 训练费 / 维护费 / 训练时间 / 训练建筑 / 拥有数量 / 被动技能
  │       ├─ 训练上限信息（驻军 / 远征 / 余量）
  │       ├─ 训练队列状态（暂停原因 / 取消队列按钮）
  │       └─ 训练操作（数量输入 + 训练 / MAX / 删除按钮）
  │
  ├─ 战斗 (fight)
  │   ├─ 远征 (expedition)
  │   │   ├─ 远征阵容编辑
  │   │   │   ├─ 前排 (front) · 1-2 格
  │   │   │   ├─ 中排 (mid)  · 1-2 格
  │   │   │   └─ 后排 (back) · 1 格
  │   │   │   └─ 每格：兵种图标 + 人数 ± / MAX / 移除 → 点击打开编队弹窗
  │   │   ├─ 使用上次阵容 / 清空阵容 按钮
  │   │   ├─ 木人桩训练按钮 → 战斗画面（训练模式）
  │   │   ├─ 关卡选择下拉 + ◀▶ 切换
  │   │   └─ 开战按钮 → 战斗画面（实战模式）
  │   │
  │   ├─ 驻军 (garrison)
  │   │   ├─ 驻军阵容编辑（同远征三排结构）
  │   │   ├─ 清空驻军 按钮
  │   │   ├─ 测试触发入侵 按钮
  │   │   └─ 驻军功勋 + 最近驻军事件日志
  │   │
  │   └─ （已移除）训练场独立子标签
  │
  ├─ 科技 (tech)
  │   └─ 科技树页面（开发中，后续开放兵种升级）
  │
  └─ 日志 (log)
      └─ 事件日志（倒序完整列表）



弹窗
  ├─ 编队弹窗（选择兵种 + 数量 + 确认编入）
  ├─ 单位详情弹窗（全属性面板）
  └─ 设置弹窗（游戏总时间 / 激活码输入 / 重置存档）

战斗画面（全屏覆盖）
  ├─ 顶部栏（标题 / 速度 1x 2x 4x / 逃跑）
  ├─ 战斗区域（敌方上方 / 我方下方 / 前中后排 / 兵团框动画）
  ├─ VFX 特效层（兵种对应特效 PNG）
  ├─ 战斗结果面板（胜利/战败/超时 + 战利品/战损 + 下一关/重新对战/退出）
  └─ 战斗消息日志（滚动 25 条）

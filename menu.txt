放置帝国 — 菜单目录
=========================================================

顶部资源栏
  ├─ 木材 (wood)
  ├─ 石料 (stone)
  ├─ 食物 (food)
  ├─ 科技点 (tech)
  └─ 村民 (pop)

底部导航
  ├─ 主页 (home)
  │   ├─ 城镇 & 村民分配卡片（合并：等级+升级条件右对齐 / 村民+仓库 / 升级按钮同行右对齐）
  │   ├─ 城镇巡防地图（建筑 / 工人动画 / 驻军守卫 / 入侵 VFX）
  │   ├─ 最近事件卡片（固定行高90px，日志+驻军事件合并倒序5条）
  │   ├─ 设置按钮（居中，点击打开设置弹窗）
  │   └─ 测试工具卡片（激活码1122解锁后显示）
  │       ├─ 一键添加木/石/食/科技（默认50000）
  │       ├─ 添加战功（默认50）
  │       ├─ 一键添加全部精魄（默认5）
  │       └─ 秒升建筑开关（开启后所有升级时间=1秒）
  │
  ├─ 建筑 (build) — 3个子标签
  │   ├─ 基础建筑 (basic)
  │   │   ├─ 营帐 (barracks) — 出战上限 5+lv×5 人/格
  │   │   ├─ 仓库 (warehouse) — 升级消耗 = 等级×upBase（线性），容量 = storageBase + lv×storagePerLv
  │   │   ├─ 伐木场 (lumber_mill)
  │   │   ├─ 采石场 (quarry)
  │   │   └─ 农田 (farm)
  │   ├─ 兵营建筑 (barracks)
  │   │   ├─ 步兵营地 (infantry_camp)
  │   │   ├─ 弓兵营地 (archer_range)
  │   │   ├─ 骑兵训练场 (stable)
  │   │   └─ 法师塔 (mage_tower) — T2起步，需击败第4个Boss解锁
  │   └─ 特殊建筑 (special)
  │       └─ 箭塔 (arrow_tower)
  │
  ├─ 军营 (barracks) — 双标签页
  │   ├─ 总兵力 / 营帐上限 / 口粮每秒消耗
  │   │
  │   ├─ 训练军队 (train)
  │   │   ├─ 4线折叠分支（步兵/弓兵/骑兵/法师），只显示最新已研究tier
  │   │   │   分支头：线名 + 当前T级 + 兵种数 + 总上限 + 余量
  │   │   │   满足条件时显示「可研究」标记（科技点/战功/精魄/资源均达标）
  │   │   │   锁定线显示🔒和解锁原因（未建建筑/未击败Boss）
  │   │   └─ 每兵种卡片（同下）
  │   │       ├─ 标题行：兵种名 + 训练建筑 + 属性按钮 | 已达上限（右对齐）
  │   │       ├─ 属性行：被动 | ATK/DEF
  │   │       ├─ 消耗行：资源/人 | 队列暂停原因 + 训练队列 + 取消队列（右对齐）
  │   │       ├─ 未解锁：红色解锁条件 → 无研究按钮/训练操作
  │   │       └─ 已解锁：训练操作（输入 + 训练 / MAX / 删除）| 上限+拥有（右对齐）
  │   │       研究到更高 tier 时自动退还旧 tier 全部兵力（pool + 编队 + 队列）
  │   │
  │   └─ 阵容方案 (formation)
  │       ├─ 远征阵容概览（前/中/后排各槽位兵种+人数）
  │       ├─ 驻军阵容概览
  │       └─ 快捷按钮：使用上次阵容 / 清空 / 跳转编队
  │
  ├─ 战斗 (fight)
  │   ├─ 远征 (expedition)
  │   │   ├─ 远征阵容编辑
  │   │   │   ├─ 前排 (front) · 随关卡解锁 1→4 格
  │   │   │   ├─ 中排 (mid)  · 随关卡解锁 1→4 格
  │   │   │   └─ 后排 (back) · 随关卡解锁 1→4 格
  │   │   ├─ 阵容解锁：每5关轮替解锁（前→中→后），第5关起，上限 3×4
  │   │   │   第5关前+1 / 第10关中+1 / 第15关后+1 / 第20关前+1
  │   │   │   第25关中+1 / 第30关后+1 / 第35关前+1 / 第40关中+1 / 第45关后+1
  │   │   ├─ 使用上次阵容 / 清空阵容 / 训练场 按钮（同行）
  │   │   ├─ 关卡选择下拉 + ◀▶ 切换（100关，10章×10关，每10关Boss）

放置帝国 — 菜单目录
====================

顶部资源栏
  ├─ 木材 (wood)
  ├─ 石料 (stone)
  ├─ 食物 (food)
  ├─ 科技点 (tech)
  └─ 人口 (pop)

底部导航
  ├─ 主页 (home)
  │   ├─ 城镇信息卡片（等级 / 人口 / 仓库 / Boss 进度 / 升级按钮）
  │   ├─ 人口分配卡片（木材/石料/食物 ± 调节 + 采集速率 + Buff%）
  │   ├─ 城镇巡防地图（建筑 / 工人动画 / 驻军守卫 / 入侵 VFX）
  │   └─ 最近事件卡片（日志倒序 6 条）
  │
  ├─ 建筑 (build)
  │   ├─ 基础建筑 (basic)
  │   │   ├─ 营帐 (barracks)
  │   │   ├─ 仓库 (warehouse)
  │   │   ├─ 伐木场 (lumber_mill)
  │   │   ├─ 采石场 (quarry)
  │   │   └─ 农田 (farm)
  │   ├─ 兵营建筑 (barracks)
  │   │   ├─ 步兵营地 (infantry_camp)
  │   │   ├─ 射手靶场 (archer_range)
  │   │   ├─ 骑兵训练场 (stable)
  │   │   ├─ 长矛兵营地 (spear_crypt)
  │   │   └─ 法师塔 (mage_tower)
  │   ├─ 军事学院 (academy)
  │   │   ├─ 军事学院 (military_academy) — 研究兵种进阶，解锁高阶兵种训练
  │   │   └─ 步兵升级树：农民→民兵→(重盾手/长矛扈从/双手剑士)→(堡垒巨盾/铁玫瑰/血蔷薇)
  │   ├─ 特殊建筑 (special)
  │   │   └─ 箭塔 (arrow_tower)
  │   └─ 英雄祭坛 (altar) · 暂无建筑
  │
  ├─ 军营 (barracks)
  │   ├─ 总兵力 / 营帐上限
  │   ├─ T0 基础 / T1 进阶 / T2 精锐 / T3 终极 选项卡
  │   ├─ T0：全部基础兵种（农民/弓兵/骑兵/长矛兵/法师）
  │   ├─ T1~T3：步兵升级线变体（民兵 → 重盾手/长矛扈从/双手剑士 → 堡垒巨盾/铁玫瑰/血蔷薇）
  │   └─ 每兵种卡片
  │       ├─ 兵种名（可点击 → 单位详情弹窗）+ 训练建筑标签 + 属性按钮
  │       │   └─ 属性面板：攻击力 / 防御力 / 速度 / 训练费 / 维护费 / 训练时间 / 训练建筑 / 拥有数量 / 被动技能
  │       ├─ 训练上限信息（驻军 / 远征 / 余量）
  │       ├─ 训练队列状态（暂停原因 / 取消队列按钮）
  │       ├─ 未解锁变体：研究按钮（消耗资源在军事学院解锁）
  │       └─ 已解锁：训练操作（数量输入 + 训练 / MAX / 删除按钮）
  │
  ├─ 战斗 (fight)
  │   ├─ 远征 (expedition)
  │   │   ├─ 远征阵容编辑
  │   │   │   ├─ 前排 (front) · 1-2 格
  │   │   │   ├─ 中排 (mid)  · 1-2 格
  │   │   │   └─ 后排 (back) · 1 格
  │   │   │   └─ 每格：兵种图标 + 人数 ± / MAX / 移除 → 点击打开编队弹窗
  │   │   ├─ 使用上次阵容 / 清空阵容 按钮
  │   │   ├─ 训练场按钮 → 战斗画面（训练模式）
  │   │   ├─ 关卡选择下拉 + ◀▶ 切换
  │   │   └─ 开战按钮 → 战斗画面（实战模式）
  │   │
  │   ├─ 驻军 (garrison)
  │   │   ├─ 驻军阵容编辑（同远征三排结构）
  │   │   ├─ 清空驻军 按钮
  │   │   ├─ 测试触发入侵 按钮
  │   │   └─ 驻军功勋 + 最近驻军事件日志
  │   │
  │   └─ （已移除）训练场独立子标签
  │
  ├─ 科技 (tech)
  │   └─ 科技树页面（科技点资源，后续扩展科技树）
  │
  └─ 日志 (log)
      ├─ 事件日志（倒序完整列表）
      ├─ 激活码输入（1122 解锁测试工具）
      ├─ 测试工具（手动添加木材/石料/食物）
      └─ 重置按钮（清除存档）



弹窗
  ├─ 编队弹窗（选择兵种 + 数量 + 确认编入）
  └─ 单位详情弹窗（全属性面板）

战斗画面（全屏覆盖）
  ├─ 顶部栏（标题 / 速度 1x 2x 4x / 逃跑）
  ├─ 战斗区域（敌方上方 / 我方下方 / 前中后排 / 兵团框动画）
  ├─ VFX 特效层（兵种对应特效 PNG）
  ├─ 战斗结果面板（胜利/战败/超时 + 战利品/战损 + 下一关/重新对战/退出）
  └─ 战斗消息日志（滚动 25 条）

放置帝国 — 菜单目录
====================

顶部资源栏
  ├─ 木材 (wood)
  ├─ 石料 (stone)
  ├─ 食物 (food)
  ├─ 科技点 (tech)
  └─ 村民 (pop)

底部导航
  ├─ 主页 (home)
  │   ├─ 城镇 & 村民分配卡片（合并：等级+升级条件右对齐 / 村民+仓库 / 升级按钮同行右对齐）
  │   ├─ 城镇巡防地图（建筑 / 工人动画 / 驻军守卫 / 入侵 VFX）
  │   ├─ 最近事件卡片（固定行高90px，日志+驻军事件合并倒序5条）
  │   ├─ 设置按钮（居中，点击打开设置弹窗）
  │   └─ 测试工具卡片（激活码解锁后显示，一键添加木/石/食/科技+战功）
  │
  ├─ 建筑 (build)
  │   ├─ 基础建筑 (basic)
  │   │   ├─ 营帐 (barracks) — 出战上限 5+lv×5 人/格
  │   │   ├─ 仓库 (warehouse) — 升级消耗 = 等级×upBase（线性），容量 = storageBase + lv×storagePerLv
  │   │   ├─ 伐木场 (lumber_mill)
  │   │   ├─ 采石场 (quarry)
  │   │   └─ 农田 (farm)
  │   ├─ 兵营建筑 (barracks)
  │   │   ├─ 步兵营地 (infantry_camp)
  │   │   ├─ 弓兵营地 (archer_range)
  │   │   ├─ 骑兵训练场 (stable)
  │   │   └─ 法师塔 (mage_tower) — T2起步，需击败第4个Boss解锁
  │   ├─ 特殊建筑 (special)
  │   │   └─ 箭塔 (arrow_tower)
  │   └─ 军事学院 (academy) — 已移除，由科技树替代
  │
  ├─ 军营 (barracks)
  │   ├─ 总兵力 / 营帐上限 / 口粮每秒消耗
  │   ├─ T0 基础 / T1 进阶 / T2 精锐 / T3 终极 / T4 传说 选项卡
  │   ├─ 按分支分组收纳（步兵线/弓兵线/骑兵线/法师线），可折叠
  │   │   └─ 分支头：兵种数 + 总上限 + 余量
  │   └─ 每兵种卡片
  │       ├─ 标题行：兵种名 + 训练建筑 + 属性按钮 | 已达上限（右对齐）
  │       ├─ 属性行：被动 | ATK/DEF/Tag（未解锁隐藏Tag）
  │       ├─ 消耗行：资源/人 | 已达上限 + 训练队列 + 取消队列（右对齐）
  │       ├─ 未解锁：红色解锁条件（去掉消耗括号）→ 无研究按钮/训练操作
  │       └─ 已解锁：训练操作（输入 + 训练 / MAX / 删除）| 上限+拥有（右对齐）
  │
  ├─ 战斗 (fight)
  │   ├─ 远征 (expedition)
  │   │   ├─ 远征阵容编辑
  │   │   │   ├─ 前排 (front) · 1-2 格
  │   │   │   ├─ 中排 (mid)  · 1-2 格
  │   │   │   └─ 后排 (back) · 1 格
  │   │   ├─ 使用上次阵容 / 清空阵容 按钮
  │   │   ├─ 训练场按钮 → 战斗画面（训练模式）
  │   │   ├─ 关卡选择下拉 + ◀▶ 切换（100关，10章×10关，每10关Boss）
  │   │   └─ 开战按钮 → 战斗画面（实战模式）
  │   │
  │   ├─ 驻军 (garrison)
  │   │   ├─ 驻军阵容编辑（同远征三排结构）
  │   │   ├─ 清空驻军 按钮
  │   │   ├─ 测试触发入侵 按钮
  │   │   └─ 驻军功勋 + 最近驻军事件日志
  │   │
  │   └─ （已移除）训练场独立子标签
  │
  ├─ 科技 (tech)
  │   ├─ 精魄库存（Boss概率掉落，用于T2/T3解锁）
  │   ├─ 科技点/战功/精魄资源概览
  │   ├─ 步兵线（农民→民兵→重盾手/长矛扈从/双手剑士→堡垒巨盾/铁玫瑰/血蔷薇）
  │   ├─ 弓兵线（猎人→游侠→银弓猎手/重弩手/双刃刺客→不列颠长弓手/热那亚劲弩/幽影刃侍）
  │   └─ 骑兵线（侍从骑士→猎风弩骑/重装骑士→破晓龙息/条顿骑士）
  │       └─ 每个节点：T级/名称/Tag/研究按钮（消耗资源+科技点+战功+精魄）
  │
  └─ 日志 (log)
      └─ 事件日志（倒序完整列表）

弹窗
  ├─ 编队弹窗（选择兵种 + 数量 + 确认编入）
  ├─ 单位详情弹窗（全属性面板）
  └─ 设置弹窗（游戏总时间 / 激活码 / 重置存档）

战斗画面（全屏覆盖）
  ├─ 顶部栏（标题 / 速度 1x 2x 4x / 逃跑）
  ├─ 战斗区域（敌方上方 / 我方下方 / 前中后排 / 兵团框动画）
  ├─ VFX 特效层（兵种对应特效 PNG）
  ├─ 战斗结果面板（胜利/战败/超时 + 战利品/战损 + 下一关/重新对战/退出）
  └─ 战斗消息日志（滚动 25 条）

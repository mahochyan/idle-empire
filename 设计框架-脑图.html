⚔ 文字挂机 RTS 游戏 — 设计框架 v3

核心循环

  • 资源生产 → 建筑升级 → 解锁
  • 训练军队 → 侦察 → 布阵 → 战术
  • 自动战斗 → 功勋/掉落 → 下一层

多重门槛

  • 资源 + 建筑等级 + 击败Boss

一、资源系统

分层资源

  • T1：木材、石料、食物 初始
  • T2：铁矿、煤炭 矿场N级
  • T3：秘银、水晶 秘法建筑N级
  • T4：龙晶、神血 特殊建筑N级

采集机制

  • 建筑按时间自动产出
  • 升级建筑提升速率
  • 新资源需投入前期资源
  • 资源间存在依赖关系

二、兵种系统

4 普通兵种 — 经典 RTS 克制环

  • 步兵（人族） 前排/格挡
  • → 克 枪兵：贴身近战
  • 弓兵（精灵） 后排/暴击
  • → 克 步兵：远程消耗
  • 骑兵（兽人） 前排/吸血
  • → 克 弓兵：冲锋切入
  • 枪兵（亡灵） 中排/破甲
  • → 克 骑兵：反冲锋

克制闭环

  • 步兵 → 枪兵 → 骑兵 → 弓兵 → 步兵

法师 — 独立体系

  • 玻璃大炮：对所有普通兵互易伤
  • 法师 → 普通兵：1.3× 伤害
  • 普通兵 → 法师：1.3× 伤害
  • 需要前排保护才能输出

种族被动

  • 人族→格挡 精灵→暴击
  • 兽人→吸血 亡灵→破甲/法伤

升级线（分期）

  • P1：4+1 基础兵种
  • P2：基础→进阶→精锐
  • P3：分支（攻/防）
  • P4：神级 2 分支
  • 总计：5 × 6 = 30 单位（含法师）

三、战斗系统

基本规则

  • 回合制，自动演算，文字播报

阵型系统

  • 前排：步兵/骑兵，优先承伤
  • 中排：枪兵，稳定输出
  • 后排：弓兵/法师，高伤脆皮
  • 前排击破 → 后排暴露 → 承伤+50%

战术指令（战前选择）

  • 稳扎稳打：防+10% 速-5%
  • 全军突击：攻+15% 承伤+10%
  • 远程压制：后排+12% 前排+8%承伤
  • 消耗战：每回合回复 回合+3

情报侦察

  • 敌方构成 / 属性 / 弱点
  • Boss 特殊技能提示
  • 建议阵容方向

四、伤害公式

公式

  • 攻击 × 技能 × 克制 × 易伤 × 战术 × 浮动 − 防御

克制倍率（普通兵种间）

  • 优势：1.35 | 普通：1.00 | 劣势：0.75

法师易伤（双向）

  • 法师 ↔ 普通兵：1.3
  • 最大倍率 1.35×1.3 ≈ 1.75

五、军营系统

营地

  • 人口容量上限（可升级）
  • 建筑：采集 / 兵营 / 防御

出战阵容

  • 独立上阵槽位（可扩容）
  • 阵型编排（前/中/后排）
  • 针对敌情调整配比

常驻战斗 (P3+)

  • Boss 防御战 + 刷怪点

六、英雄系统 (P3+)

  • 独立成长线
  • 带队参战 / 全局增益
  • 特性 + 装备 + 任务
  • 等待期核心养成

七、节奏控制

建筑升级时长

  • 1→2：5min | 3→4：30min
  • 5→6：2h

人口限制

  • 高级兵消耗更多人口

等待期可做

  • 刷低层 | 侦察 | 调阵容
  • 营地事件 | 养英雄

八、UI 布局

多标签页

  • 资源 | 建筑 | 军营 | 阵容
  • 战斗 | 英雄 | 日志

战斗页

  • 敌情 → 阵容+战术 → 开战
  • 战斗日志逐行播报
  • 结果奖励

九、开发阶段

Phase 1 — 最小可玩

  • 3 资源：木材、石料、食物
  • 3 建筑：伐木场、采石场、农田
  • 4 普通兵种：步/弓/骑/枪
  • +1 特殊：法师（互易伤）
  • 克制环 + 阵型 + 战术 + 侦察
  • 5 敌人 + 1 Boss
  • 回合战斗 + 文字播报
  • localStorage 存档

Phase 2

  • T2 资源 + 兵种升级线
  • 营地扩容 + 出战槽位

Phase 3

  • T3 资源 + 兵种分支 + 英雄
  • 营地事件

Phase 4+

  • T4 资源 + 神级双分支

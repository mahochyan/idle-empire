# 文字挂机RTS游戏 - 设计框架 v3

---

## 一、游戏概述

- 类型：文字挂机 RTS，回合制战斗
- 平台：Web 技术（HTML/CSS/JS），安卓浏览器/WebView 运行
- 存档：localStorage 本地存储
- 风格：资源驱动军备，RTS 指挥感，纯文字界面

### 核心循环

```
城镇等级 → 人口上限 → 人口分配采集资源
    ↓
建筑升级(效率Buff) → 资源产出加速
    ↓
训练军队 → 驻军防御(日常侵袭) + 远征出战(选择关卡→布阵→战斗)
    ↓
获得功勋/掉落 → 击败Boss → 城镇可升级 → 下一大关
```

## 二、城镇与人口系统（核心大本营）

### 2.1 城镇等级

城镇是整个游戏的大本营，决定人口上限。击败守关 Boss 后可升级。

| 城镇等级 |   名称 | 人口上限 |   升级条件   |
|----------|--------|---------|--------------|
|   Lv.1   | 小村庄 |    10   | 初始         |
|   Lv.2   | 村庄   |    20   | 击败第 10关  |
|   Lv.3   | 大村庄 |    30   | 击败第 20关  |
|   Lv.4   | 小镇   |    40   | 击败第 30关  |
|   Lv.5   | 城镇   |    50   | 击败第 40关  |
|   Lv.6   | 大城镇 |    60   | 击败第 50关  |
|   Lv.7   | 小城   |    70   | 击败第 60关  |
|   Lv.8   | 大城市 |    80   | 击败第 70关  |
|   Lv.9   |巨型城市|    90   | 击败第 80关  |
|   Lv.10  | 大都会 |    100  | 击败第 90关  |

> 前期每关可升一级，后期需要跨越多关才能升级，形成长线目标。

### 2.2 人口分配采集

- 人口是资源采集的唯一来源，不是建筑
- 玩家将人口分配到各资源类型上工作
- 每人每秒基础产出固定（如 0.5/秒），建筑效率倍率加成
- 可随时调整人口分配比例
- 未分配人口不产生任何资源

## 三、资源系统

### 3.1 分层资源

| 层级 | 资源 | 解锁前提 | 解锁内容 |
|------|------|----------|----------|
| T1 | 木材、石料、食物 | 初始可用 | 基础兵种、初级建筑 |
| T2 | 铁矿、煤炭 | 矿场 N 级 | 中级兵种、阵型扩展 |
| T3 | 秘银、水晶 | 秘法建筑 N 级 | 高级兵种、法师解锁 |
| T4 | 龙晶、神血 | 特殊建筑 N 级 | 神级兵种 |

### 3.2 采集公式

```
实际产出/秒 = 分配人口 × 基础采集率(0.5/秒) × (1 + 建筑效率Buff)
```

- 分配人口：玩家在该资源上投入的劳动力数量
- 基础采集率：每人每秒产出基准值，所有资源类型相同
- 建筑效率Buff：对应采集建筑等级提供的倍率加成

示例：
```
伐木场 Lv.3（Buff +40%），分配 5 人口到木材
木材产出 = 5 × 0.5 × 1.4 = 3.5/秒
```

### 3.3 建筑角色变化

建筑不再直接产出资源，改为提供效率倍率：

| 建筑 | 作用 | Buff 曲线（Lv.1→Lv.10） |
|------|------|------------------------|
| 伐木场 | 木材采集效率 | +10% → +200% |
| 采石场 | 石料采集效率 | +10% → +200% |
| 农田 | 食物采集效率 | +10% → +200% |
| 矿场 | 铁矿采集效率 | +10% → +200% |

- 升级建筑提升效率倍率
- 建筑之间有依赖关系（T2 建筑需消耗 T1 资源建造）

## 四、兵种系统

### 4.1 兵种体系总览

```
5 兵种（4 普通 + 1 法师）：

步兵（前排均衡）── 克枪弓骑步（1.3× 四系均势）
弓兵（后排远程）── 克步枪（1.3×），惧骑兵（无克制+50%MISS）
骑兵（前排突击）── 克弓(1.5×)克步(1.3×)，惧枪(0.7×)
枪兵（前排反骑）── 克骑兵（1.5× 反冲锋）
法师（后排高伤）── 对所有兵互易伤（1.3× 双向），ATK 15 / DEF 8
```

### 4.2 四基础普通兵种 + 克制闭环

| 编号 | 名称 | 种族 | 定位 | 攻击类型 |
|------|------|------|------|----------|
| A | 步兵 | 人类 | 前排/均衡 | 物理近战 |
| B | 弓兵 | 精灵 | 后排/远程 | 物理远程 |
| C | 骑兵 | 兽人 | 前排/突击 | 物理冲锋 |
| D | 枪兵 | 人类 | 前排/反骑 | 穿刺伤害 |

**克制倍率（实际实现）：**

```
步兵 克 枪/弓/骑/步（1.3×，全面均衡）
弓兵 克 步/枪/弓（1.3×），打骑兵无克制（1.0×）
骑兵 克 步兵(1.3×)，克弓兵(1.5×冲锋切入)，被枪兵克制(0.7×)
枪兵 克 骑兵（1.5×反冲锋），其余无克制
法师 对所有兵互易伤（1.3×，双向）
```

- **步兵 → 克枪弓骑**：全面前排战士，四系均势
- **枪兵 → 克骑兵**：长枪反冲锋，1.5× 最高克制
- **骑兵 → 克弓兵**：高速冲锋切入后排，1.5×
- **骑兵 → 惧枪兵**：反冲锋被克，0.7×
- **弓兵 → 克步枪**：远程消耗近战单位
- **弓兵 → 惧骑兵**：无克制加成，且 MISS 率 50%

### 4.3 法师 — 独立体系

法师不在四兵种克制环内，有独立的互易伤机制：

| 特性 | 说明 |
|------|------|
| 法师 → 普通兵 | 伤害 ×1.3 |
| 普通兵 → 法师 | 伤害 ×1.3 |
| 法师 ↔ 法师 | 伤害 ×1.3（代码实现为统一1.3） |

> 定位：玻璃大炮。高输出（ATK 15）但极度脆弱（DEF 8），需要前排保护。双方互有加成，骑兵切后排法师收益极高。

### 4.4 兵种被动

| 兵种 | 被动 | 效果 |
|------|------|------|
| 步兵 | 攻击增强 | ATK × 1.1 |
| 弓兵 | 远程闪避 | 基础 20% MISS；打骑兵 50% MISS |
| 骑兵 | 闪避 | 对非弓兵攻击有 10% 概率闪避 |
| 枪兵 | 暴击 | 10% 概率双倍伤害 |
| 法师 | 互易伤 | 对普通兵伤害 ×1.3，被普通兵打 ×1.3 |

### 4.5 升级线（分期实现）

| 阶段 | 内容 |
|------|------|
| Phase 1 | 4 基础普通兵种 + 法师基础形态 |
| Phase 2 | 每个兵种 1 条升级线（基础→进阶→精锐） |
| Phase 3 | 每个兵种分支（攻击/防御路线） |
| Phase 4 | 神级单位（双分支） |

神级分支设计（预留）：
- 步兵 → 神·圣殿骑士（攻）/ 神·铁壁守卫（防）
- 弓兵 → 神·风行者（攻）/ 神·猎命者（防）
- 骑兵 → 神·毁灭骑士（攻）/ 神·不朽铁骑（防）
- 枪兵 → 神·龙枪将（攻）/ 神·不屈之壁（防）
- 法师 → 神·大魔导（攻）/ 神·死灵王（防）

## 五、战斗系统

### 5.1 基本规则

- 回合制，自动演算
- 阵型系统：前排/中排/后排
- 战术指令：默认稳扎稳打（UI 暂隐藏）
- 文字播报战斗过程

### 5.2 阵型系统

```
前排：近战必须在前排才能攻击，前排被击破后中排自动补位
     步兵、骑兵、枪兵 适合
中排：过渡位置，前排空缺时自动前移
后排：远程/法术可任意排攻击
     弓兵、法师 适合
```

- 近战单位（步兵/骑兵/枪兵）只有在前排才能攻击目标
- 前排全灭后，中排单位整体前移补位（shiftRows），后排→中排同理
- 远程单位（弓兵/法师）可在任意排攻击
- 阵型槽位：初始 1×3，击败 Boss 后扩展（Boss1→前排2格）

### 5.3 战术指令

战斗默认使用"稳扎稳打"战术（UI 入口已隐藏，归入后续英雄技能系统）。

| 战术 | 效果（config.js 实际值） |
|------|------|
| 稳扎稳打 | 防御 +15%，速度 -10% |
| 全军突击 | 攻击 +20%，防御 -10% |
| 远程压制 | 后排伤害 +15%，前排防御 -5% |
| 消耗战 | 防御 +5%，速度 -5%，每回合回复 5% |

### 5.4 情报侦察（Phase 2+ 预留）

当前版本直接在关卡选择界面显示敌方编队构成，无独立侦察步骤。
规划中的侦察系统：敌方兵种/属性/弱点提示 + Boss 特殊技能预警。

### 5.5 伤害公式（实际实现）

```
damage = hp × atk × 0.09 × (100 / (100 + def × 8)) × counter × mage × passive × random(0.9~1.1)
```

- **hp × atk**：参战人数 × 攻击力（人数越多伤害越高）
- **0.09**：全局伤害系数
- **100/(100+def×8)**：防御减免因子（高防单位承伤大幅降低）
- **counter**：克制倍率（1.5 / 1.3 / 1.0 / 0.7）
- **mage**：法师互易伤系数（1.3×，双向）
- **passive**：兵种被动（步兵 ATK×1.1；枪兵暴击 10%→2× 伤害）
- **random**：±10% 随机浮动

弓兵 MISS 和骑兵闪避在命中判定阶段处理，不在伤害公式内。

## 六、军营系统

### 6.1 训练建筑

每种兵种有独立训练建筑，等级决定后备上限：

| 建筑 | 兵种 | 后备上限（Lv.1→Lv.9） |
|------|------|----------------------|
| 步兵营地 | 步兵 | 4→18 |
| 射手靶场 | 弓兵 | 3→17 |
| 骑兵训练场 | 骑兵 | 2→9（击败1 Boss解锁） |
| 长矛兵营地 | 枪兵 | 2→9（击败2 Boss解锁） |
| 法师塔 | 法师 | 2→9（击败2 Boss解锁） |

- 营帐(barracks)控制出战/驻军每格人数上限（固定 100 人/格）
- 训练队列：资源在训练完成时扣除，每秒生产 1 人
- 队列暂停：资源不足或后备满时自动暂停并显示原因

### 6.2 出战阵容

- 独立于后备池的上阵槽位，初始 1×3（前/中/后各1格）
- 击败 Boss 后扩展槽位：Boss1→前排2格，后续逐级扩展
- 每格上限 100 人
- 编队弹窗：选兵种 → 定人数（长按加减/MAX 一键填满）

### 6.3 驻军阵容

- 独立于出战阵容的防御编队，显示在主页城镇巡防地图上
- 与出战共享兵种人数池（同一批兵不能同时出战+驻军）
- 驻军阵容不影响日常战斗，用于后续防御侵袭系统

## 七、驻军系统 — 日常防御怪物侵袭（Phase 2+）

> **当前实际实现**：驻军编队 UI 已完成（`rGarrison()`），玩家可编排驻军阵容并显示在主页城镇巡防地图上。
> 以下为完整的日常防御侵袭系统设计，尚未实装。

### 7.1 系统定位

```
驻军系统 = 城镇被动防御 + 日常怪物侵袭 + 闲置兵力利用
```

- 玩家在城镇驻扎部分兵力，用于防御周期性怪物侵袭
- 侵袭自动演算（或可选手动观战），不强制操作
- 闲置在军营池的兵力自动参与防御，不占用出战阵容
- 成功防御获得资源/功勋/稀有掉落，失败则资源被掠夺

### 7.2 侵袭机制

| 要素 | 说明 |
|------|------|
| 触发方式 | 日常定时（如每15分钟一波），或累计在线时长触发 |
| 怪物强度 | 随城镇等级递增，随机兵种组合（步/弓/骑/枪/法混编） |
| 波次 | 每轮 1~3 波，Boss 周（每击败 N 个主线 Boss）出现精英侵袭 |
| 预警 | 侵袭前 2 分钟 UI 提示"怪物正在接近…"，玩家可调兵增援 |
| 掠夺 | 防御失败时，损失当前库存资源的 10%~20% |

### 7.3 驻军配置

```
驻军阵容 = 独立于出战阵容的防御编队
```

- **驻军槽位**：初始 1×3（前1/中1/后1），随城镇等级扩展
- **兵力来源**：从军营后备池（S.pool）分配士兵到驻军阵容
- **驻军限制**：与出战阵容共享兵种人数池（同一批兵不能同时出战+驻军）
- **自动补员**：驻军伤亡后，后备池有剩余时自动补充（需手动开启）
- **驻军战术**：独立战术指令（默认"坚守阵地"——防御+20%，速度-10%）

### 7.4 防御战斗规则

- 自动演算，使用与主线相同的战斗引擎
- 回合制，驻军为"我方"，怪物为"敌方"
- 交替攻击制（与主线一致）
- 驻军全灭 → 防御失败，怪物掠夺资源
- 怪物全灭 → 防御成功，获得奖励

### 7.5 奖励体系

| 奖励类型 | 来源 | 说明 |
|---------|------|------|
| 基础资源 | 每波防御成功 | 木材/石料/食物，量随怪物强度递增 |
| 功勋 | 每轮侵袭全清 | 累计功勋解锁驻军专属建筑/升级 |
| 稀有掉落 | Boss 侵袭 | 装备材料、英雄碎片（P3）、加速道具 |
| 侵袭宝箱 | 每日首轮 | 保底奖励，鼓励每日登录 |

### 7.6 驻军建筑（预留）

| 建筑 | 效果 | 解锁 |
|------|------|------|
| 瞭望塔 | 侵袭预警提前 1 分钟 | 城镇 Lv.2 |
| 城墙 | 驻军防御 +15% | 城镇 Lv.3 |
| 陷阱工坊 | 每波开始前对怪物造成 5% 兵力伤害 | 城镇 Lv.4 |
| 烽火台 | 侵袭间隔 +5 分钟（减少频率） | 城镇 Lv.5 |
| 军械库 | 驻军攻击 +10% | 击败 3 个 Boss |

### 7.7 设计意图

- **闲置兵力有价值**：后备池的兵不再只是"等出战"，还有防御用途
- **日常登录动力**：每日首轮侵袭宝箱 + 定时防御奖励
- **资源回收口**：防御失败掠夺资源，防止资源无限囤积
- **策略深度**：驻军 vs 出战兵力的分配权衡

## 八、英雄系统（Phase 3+）

### 8.1 核心定位

```
英雄 = 方阵核心单位 + 技能载体 + 装备成长 + 阵容流派发动机
```

- 英雄占 1 个出战槽位，站在前/中/后排某位置
- 本身是战斗单位，有血量/攻击/防御，不是全局挂件
- **每场战斗只能上阵 1 个英雄**

### 8.2 主动技能

每个英雄带 1~2 个主动技能，战斗中回合触发：

| 技能类型 | 例子 |
|---------|------|
| 指挥类 | "全军冲锋"：本回合所有前排伤害 +20% |
| 伤害类 | "暴风雪"：对敌方后排造成范围伤害 |
| 控场类 | "盾墙"：本回合前排承伤 -30% |
| 治疗类 | "战场包扎"：回复我方血量最低单位 |
| 针对类 | "斩首"：跳过前排直击敌方后排 |

### 8.3 装备系统

英雄有独立装备栏（武器/防具/饰品），影响技能效果和属性。

### 8.4 流派发动机

英雄决定阵容流派方向。同一套兵种，不同英雄打出完全不同效果：

| 英雄类型 | 流派方向 |
|---------|---------|
| 骑兵统帅型 | 骑兵数量越多，冲锋伤害越高 → 骑突流派 |
| 步阵防守型 | 步兵承伤时概率反击 → 铁壁反击流 |
| 弓兵指挥型 | 后排弓兵集火优先击杀 → 狙杀流 |
| 法师增幅型 | 法师易伤倍率从 1.3 提到 1.6 → 玻璃大炮流 |
| 混合型 | 场上每有一种兵种，全体 +5% → 万金油流 |

### 8.5 决策链

```
侦察情报 → 选择英雄(流派方向) → 搭配兵种 → 编排阵型 → 选择战术 → 开战
```

## 十、数值设计（当前实际值）

- 伤害公式：`hp × atk × 0.09 × (100/(100+def×8)) × counter × mage × passive × random(0.9~1.1)`
- 克制倍率：1.5（枪→骑、骑→弓）/ 1.3（大部分克制）/ 1.0 / 0.7（骑→枪）
- 法师互易伤（双向）：1.3
- 暴击（枪兵 10%）：2× 伤害
- 弓兵 MISS：基础 20%，对骑兵 50%
- 骑兵闪避：对非弓兵 10%
- 步兵被动：ATK × 1.1
- 所有兵种、资源、建筑数据配置化（config.js + levels.js）
- 各系统模块化，独立扩展互不影响

## 十三、长期游玩设计

### 13.1 设计原则

游戏不是限时运营型，而是单机永久可玩型。通过以下系统层层叠加，让玩家在单机环境下持续获得成长目标。

```
主线推进 → 远征篇章 → 英雄装备 → 挑战词条 → 周目轮回
   ↑                                              │
   └──────────── 传承加成 ← 新一轮更强 ──────────────┘
```

### 13.2 远征篇章

远征是永久开放内容，不设限时。和主线的区别：

| | 主线 | 远征 |
|------|------|------|
| 敌人 | 固定配置 | 独立敌人 + 地形词条 |
| Boss | 每层 1 个守关 | 独特 Boss 机制 |
| 奖励 | 解锁下一层 | 专属奖励（装备、英雄碎片） |
| 重置 | 不重置 | 可重复挑战 |
| 开放 | 依次解锁 | 达成条件后永久开放 |

每个远征拥有：
- 独立敌人编队（可能混合不常见兵种组合）
- 地形词条（如"山地：骑兵速度 -20%"、"沼泽：步兵防御 -10%"）
- Boss 特殊机制（如"每回合召唤 2 个小兵"、"血量低于 50% 时狂暴"）
- 专属掉落（远征限定装备、英雄解锁碎片）

### 13.3 周目轮回

玩家通关一个大阶段后，可选择"进入新纪元"。

**轮回保留：**
- 英雄（含等级、技能、装备）
- 兵种图鉴（已解锁的兵种永久记录）
- 永久天赋/传承加成
- 部分特殊装备

**轮回重置：**
- 资源清零
- 建筑等级重置
- 普通兵种数量清零

**传承点：**
- 根据轮回前进度结算传承点
- 传承点用于解锁永久加成（资源产出 +X%、兵种属性 +X% 等）
- 多周目累加，越轮回越强

### 13.4 挑战词条

玩家在战斗/远征中可主动选择额外难度词条，挑战与奖励成正比。

**词条示例：**

| 类型 | 词条 | 效果 | 奖励加成 |
|------|------|------|----------|
| 敌人强化 | 狂化 | 敌方攻击 +20% | +15% |
| 敌人强化 | 钢铁之躯 | 敌方防御 +30% | +15% |
| 我方限制 | 物资紧缺 | 出战兵力上限 -2 | +20% |
| 我方限制 | 疲兵作战 | 我方速度 -15% | +15% |
| 环境词条 | 暴风雨 | 远程伤害 -25% | +20% |
| 环境词条 | 诅咒之地 | 每回合全体扣血 3% | +25% |
| 机制词条 | 孤军奋战 | 不可使用同种兵超过 1 个 | +30% |

> 玩家可叠加多个词条，倍率累加。高风险高回报。

### 13.5 资源回收（长期消耗口）

避免后期低级资源无用，每个层级资源都有持续消耗出口：

| 资源 | 消耗出口 |
|------|---------|
| T1 木材/石料/食物 | 兵种训练消耗、营地维护、远征补给 |
| T2 铁矿/煤炭 | 高级兵维护费、装备锻造材料、建筑加速 |
| T3 秘银/水晶 | 英雄装备强化、神级单位前置、传承解锁 |
| T4 龙晶/神血 | 神级兵种训练、周目轮回保留道具、终极装备 |

核心原则：
- **低级资源永不过期**：T1 到后期仍然需要大量消耗
- **资源可向上转换**：比例递减（如 100 木材 → 1 铁矿）
- **高级资源稀缺**：T3/T4 无法靠低级转换大量获得

### 战斗回合规则（2026-05-12 重做）

- **交替攻击制**：双方各自按速度降序排列（同速50%随机），先手方由最快单位决定
- **严格轮替**：大阵容不能靠人数多获得更多攻击次数
- **小阵容循环**：人数少的一方用完所有单位后从头循环，保持双方攻击次数对等
- **Boss 战例外**：单 Boss 时我方全员按速度顺序先攻，Boss 后攻
- **训练场**：空敌方时我方按速度顺序攻击，木人桩不还手
